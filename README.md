# 琵琶调音器

这是一个面向 iPhone 的极简琵琶调音辅助应用，当前第一版目标是：

- 中文界面
- iOS 16+
- 用户手动选弦
- 支持四弦普通成人琵琶定弦
- 用麦克风拾音并给出偏高、偏低、接近标准的反馈

## 当前定弦

从四弦到一弦：

- 四弦（缠弦）：A2，220 Hz，低音 5
- 三弦（老弦）：D4，294 Hz，中音 1
- 二弦（中弦）：E4，330 Hz，中音 2
- 一弦（子弦）：A4，440 Hz，中音 5

## 目录结构

- `PipaTuner.xcodeproj`：Xcode 工程
- `PipaTuner/App`：应用入口
- `PipaTuner/Models`：定弦与调音结果模型
- `PipaTuner/Services`：麦克风采集与音高识别
- `PipaTuner/Views`：极简中文界面
